# My-C-Learning
仓库简介
本仓库为个人C语言入门学习的全程记录，从基础语法到简单小项目，包含每日练习代码、知识点实操、问题调试过程，纯个人手写练习，用于复盘学习轨迹、积累编程基础，也为同阶段入门者提供参考。
持续更新中，跟随C语言学习进度逐步完善～
仓库结构
C-Learning-Notes/
├── 01_基础语法/       // 变量、数据类型、输入输出、运算符等基础练习
├── 02_循环与分支/     // if/switch、for/while/do-while、循环嵌套等实操
├── 03_数组与字符串/   // 一维/二维数组、字符串处理、数组相关算法练习
├── 04_函数与递归/     // 函数定义调用、参数传递、递归实现、库函数使用
├── 05_指针与结构体/   // 指针基础、指针与数组、结构体定义与使用
├── 06_文件操作/       // 文件打开/读写/关闭、文件相关函数实操
├── 07_小项目实战/     // 基于所学的综合小项目
├── notes/             // 学习笔记，记录知识点、易错点、调试思路
└── README.md          // 仓库说明文档
